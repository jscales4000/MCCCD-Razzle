Reflected Ceiling Routing

Three takes on the reflected-ceiling-plan routing concept. Each mockup is rendered at the panel's native 1280×800, in the active MCCCD orange theme. Tap a display in the room layout to choose its source — the three mockups differ in how that picker appears.

Mockup A
Schematic + Modal

Clean ceiling plan as the canvas. Tap a display, centered source picker drops over the layout. Most explicit — the picker stays out of the way until needed.

Centered Modal Plan View Quick Routes
Mockup B
Live Map + Inline Popover

Displays show current source inline as a live badge. Sidebar mirrors status. Popover anchors to the tapped display — keeps focus on context, faster to glance.

Inline Popover Status Sidebar Live Readout
Mockup C
Isometric + Bottom Sheet

3D-ish room view with perspective floor. Projection screens stand on the front wall, beams visualize the projector → screen path. Picker slides up from bottom as a sheet.

Bottom Sheet Isometric View Beam Visuals
